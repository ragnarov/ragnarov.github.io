Redundancy theorem a.k.a Consensus Theorem is a Boolean algebra trick used in Electronics.
Conditions :
  1. Expression must have three variables.
  2. Each variable is repeated twice.
  3. One variable is complemented
Action : Eliminate terms that don't include the complemented variable.
Examples :
Y= AB + AC + BC
= AB + AC
Here the term BC was eliminated.
Y= (A+B) . (A+C) . (B+C)
= (A+B) . (A+C)
Here the term (B+C) was eliminated.
When all the variables are complemented but one is NOT complemented then Eliminate the term that does not contain the NOT Complemented variable.
For instance :
Y= A B + AC + B C
= A B + AC
Here the term B C was eliminated.
